# sample-maven-app

A minimal Maven Java example project with a simple App class and JUnit 5 tests.

Quickstart
1. Build:
   mvn -v        # optional: check Maven and Java version
   mvn clean package

2. Run:
   java -jar target/sample-maven-app-0.1.0-SNAPSHOT.jar
   # or run with an argument:
   java -cp target/sample-maven-app-0.1.0-SNAPSHOT.jar com.example.App Ana

3. Run tests:
   mvn test

Create a GitHub repo and push:
1. Initialize and commit locally:
   git init
   git add .
   git commit -m "Initial commit - sample Maven app"

2. Add remote and push (SSH example):
   git remote add origin git@github.com:USERNAME/REPO.git
   git branch -M main
   git push -u origin main

Replace USERNAME/REPO with your GitHub account and repository name.
